# Paydunya payments configurations

_In order to setup mobile payments on this project, you'll need to create a `payments/.env` to store __paydunya development API credentials__ just like below._

* K_MASTER=Your_API_Master_Key
* K_PUBLIC=Your_API_Public_Key
* K_PRIVATE=Your_API_Private_Key
* Token=Your_API_Token

###### It's recommended to use your test key during development period, you should use live public_key, private_key and token in the deployed app
