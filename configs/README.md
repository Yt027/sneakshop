# Database conigurations

_In order to successfully connect this project to the sneakshop database which I assume you already created ([See main README.md file](../README.md)), you need to create a `configs/.env` file to store database and email credentials as below._
     

* DB_HOST=Your_Database_Host_Name # localhost in development mode
* DB_NAME=Your_Database_Name
* DB_USER=Your_Databse_Username
* DB_PASS=Your_Database_Password

* MAIL_HOST=Your_Mail_SMTP_Host_Address
* MAIL_KEY=Your_Mail_Application_Password
